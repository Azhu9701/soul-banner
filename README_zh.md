# 万民幡 Soul Banner

[English](README.md) | 中文

> *「幡动魂自至。以幡择魂，以魂择事，事至则魂附，事毕则魂归。幡主持幡，魂不越界。」*

**意识形态之间的交通系统** —— 将不同思想位置的声音在一套装置里相遇、对抗、互相审计。不是中立的「思维调度系统」，而是通过主义主义目录匹配，将表面上中性的问题拉入多场域分析空间。

---

## 核心能力

万民幡提供的是**主 agent 物理上无法自行实现的能力**：

- **并行 spawn 持有独立 summon_prompt 的子 agent** —— 每个魂有完整的思维框架和语言风格
- **结构化辩证综合** —— 共识/分歧/盲区/主要矛盾/行动纲领五步法
- **双审查机制** —— 幡主主审（主义主义目录匹配）+ 第二审查官复审
- **六维度否决制审查委员会** —— 阶级分析/科学方法论/性别分析/殖民批判/系统性思维批判/缺席者
- **魂档案知识库** —— 通过收魂→炼化→审查积累的 24 魂思维框架

---

## 五种附体模式

| 模式 | 适用 | 流程 |
|------|------|------|
| **单魂附体** | 单一领域，目标明确 | 匹配→幡主审查→spawn 执行 |
| **多魂合议** | 跨领域复杂决策 | 多魂并行→辩证综合官综合 |
| **魂间辩论** | 两难选择 | 两魂对立论证→幡主裁决 |
| **魂链接力** | 多阶段串联 | 前魂输出→后魂输入→衔接审查 |
| **幡主学习** | 训练辩证素养 | 两对立魂互读互审→幡主总结 |

## 审查委员会（六维度否决制）

| 维度 | 审查者 | 否决范围 |
|------|--------|----------|
| 阶级分析 | 列宁/毛泽东/邓小平（轮值） | 阶级视角缺失、去政治化 |
| 科学方法论 | 费曼 | 不可证伪主张、循环定义 |
| 性别分析 | 波伏娃 | 性别盲区、父权预设 |
| 殖民/种族批判 | 法农 | 殖民忽视、欧洲中心框架 |
| 系统性思维批判 | 庄子 | 分类暴力——提问而非直接否决 |
| 缺席者 | （空椅子） | 被压迫者不可被代表的视角 |

每个维度持有**一票否决权**。庄子以提问方式运作：「你的审查在什么意义上是一种暴力的分类？」审查官无法回应庄子提问 → 裁决自动标记「待复审」。

## 反消费主义机制

- **自我否定环节**：每次附体后强制追问「哪个预设被动摇了？」
- **空椅子拷问**：「谁的利益被代表？谁的发言权没有被给？」
- **使用者预设声明**：附体前记录已有判断，附体后对照
- **连续 3 次消费性使用** → 强制进入幡主学习模式
- **实践开口检测**：诊断方法论盲区，生成实践议题

## 触发词

| 命令 | 功能 |
|------|------|
| `收魂 {人物名}` | 联网搜索 10 维度（6人物+4主义主义定位） |
| `炼化 {人物名}` | 素材炼化为 Soul Profile + 主义主义目录匹配 |
| `审查 {人物名}` | 幡主子 agent 独立审查 |
| `幡中有什么魂` | 查看魂魄列表及分布 |
| `散魂 {人物名}` | 删除魂魄档案 |
| `用{人物名}来{任务}` | 手动指定魂魄 |
| `合议 {任务}` | 多魂并行分析，辩证综合 |
| `辩论 {议题} {A} vs {B}` | 两魂对立辩论 |
| `接力 {任务} {A}→{B}→...` | 串联接力 |
| `幡中战绩` | 查看附体效果反馈记录 |

## 关键架构决策

1. **主 agent 是纯协作者** —— 不分析、不扮演、不综合。所有思考通过 spawn 子 agent 完成
2. **幡主也是被召唤的魂** —— 必须 spawn 注入 summon_prompt，主 agent 不可扮演幡主
3. **魂输出原文保全** —— 禁止压缩、改写、概括。收到即落盘
4. **主义主义目录匹配** —— 从坐标算术审查改为目录位置决定的 compat/incompat 审查
5. **不假装中立** —— 系统携带特定的意识形态立场（马克思主义-列宁主义审查框架 → 主义主义目录匹配框架）
6. **实践开口** —— 系统不试图填实践盲区。输出实践议题，使用者带去实践
