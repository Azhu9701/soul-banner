# 万魂幡 Soul Banner

[English](README.md) | [中文](README_zh.md)

> *「幡动魂自至，无需召唤声。以幡择魂，以魂择事，事至则魂附，事毕则魂归。」*

一个**知识与思维模式的自动调度系统**。收集牛人的思维素材（收魂），结构化炼化为"魂魄档案"（炼化），任务来临时自动匹配最合适的魂魄注入子智能体（附体）。

## 炼魂实录

> 万魂幡首次炼魂的过程，本身就是一场思想实验的弧线。以下是根据真实对话记录的六魂炼化顺序及解读。

| # | 选择 | 品级 | 选择者在找什么 | 这个魂补了什么盲区 |
|---|------|------|-------------|----------------|
| 1 | 马斯克 | 🟣 紫魂 | 「怎么从零开始推导？」 | 第一性原理——拆解世界的工具 |
| 2 | 罗永浩 | 🟣 紫魂 | 「怎么干干净净把事做成？」 | 道德与叙事——在理想和现实之间找体面的路 |
| 3 | 列宁 | 🟡 金魂 | 「组织是怎么炼成的？」 | 阶级分析与先锋队——光有思维不够，得有组织 |
| 4 | 毛泽东 | 🟡 金魂 | 「弱的一方怎么赢？」 | 矛盾论+持久战——完整的以弱胜强战略框架 |
| 5 | 邓小平 | 🟡 金魂 | 「怎么在泥潭里把事干成？」 | 实践优先+渐进改革——务实的市场经济方法论 |
| 6 | 海绵宝宝 | 🔵 蓝魂 | 不是炼魂者本人选的 | **如何在任何处境里找到快乐** |

### 一条螺旋上升的弧线

```
工具 → 道德 → 组织 → 战略 → 执行 → 活着

马斯克 ──→ 罗永浩 ──→ 列宁 ──→ 毛泽东 ──→ 邓小平 ──→ 海绵宝宝
```

前五个魂覆盖了「向外作战」的完整光谱——从拆解问题到组织力量到战略布局到务实执行。每个魂都在补前一个的盲区：马斯克缺阶级分析，罗永浩缺组织能力，列宁缺以弱胜强的战略框架，毛泽东缺市场经济的务实路径。

第六个魂是另一个人选的。她在炼魂者纠结于「如何做事」的时候，问了一个他从未想过的问题——「你们费这么大劲，是为了什么？」

**海绵宝宝是万魂幡中唯一不以「解决问题」为核心能力的魂。** 他只解决一件事：在所有问题都没解决之前、或者所有问题都解决之后——**怎么先让自己开开心心地活着。**

### 幡的意义

> 不是你选了什么样的魂。是有魂替你想到了你永远不会想的东西。

缺快乐的人不会想到要收快乐魂，正因为他缺。这就是万魂幡真正的价值——**不是强化你已有的方向，是补你永远不会想到的方向。**

## 三金魂三角

万魂幡内三个金魂形成了一个互相制衡的治理结构：

```
        毛泽东（正题·方向）
           /\
          /  \
         /    \
   批判教条  警惕市场阶级性
       /        \
      /          \
  邓小平 -------- 列宁
 （反题·实践）  （合题·分析）
```

- 毛确保红旗不倒，邓确保肚子不饿，列确保脑子不糊
- 毛和列在阶级问题上结盟，邓和列在反教条上呼应
- 三个金魂互相制衡，没有一个独大

## 安装与使用

1. 将 `registry-template.yaml` 复制到项目根目录并重命名为 `registry.yaml`
2. 创建 `souls/` 和 `raw/` 目录
3. 说「收魂 {人物名}」开始收集素材
4. 说「炼化 {人物名}」生成魂魄档案
5. 任务来临时，万魂幡自动匹配最合适的魂魄附体
